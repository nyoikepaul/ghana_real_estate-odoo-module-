# Contributing to Ghana Real Estate Odoo Module

Thank you for your interest in contributing to this project! This document outlines the process for contributing to the Ghana Real Estate Odoo Module.

## How to Contribute

### 1. Fork the Repository
1. Go to [GitHub Repository](https://github.com/nyoikepaul/ghana_real_estate)
2. Click the **Fork** button in the top-right corner
3. Select your GitHub account as the fork destination

### 2. Clone Your Fork
```bash
git clone https://github.com/YOUR-USERNAME/ghana_real_estate.git
cd ghana_real_estate
```

### 3. Create a Branch
```bash
git checkout -b feature/your-feature-name
```

### 4. Make Your Changes
- Follow the existing code style
- Add comments for complex logic
- Test your changes thoroughly

### 5. Commit Your Changes
```bash
git add .
git commit -m "Add your feature description"
```

### 6. Push to GitHub
```bash
git push origin feature/your-feature-name
```

### 7. Create a Pull Request
1. Go to your fork on GitHub
2. Click **New Pull Request**
3. Select the branch you want to merge
4. Fill in the PR template
5. Submit your Pull Request

## Code Standards

### Python Code
- Follow PEP 8 guidelines
- Use meaningful variable names
- Add docstrings to functions
- Keep functions focused and concise

### XML Views
- Use proper indentation
- Follow Odoo naming conventions
- Use descriptive IDs for records

### CSS/SCSS
- Use semantic class names
- Follow BEM naming convention
- Keep styles modular

### JavaScript
- Use modern ES6+ syntax
- Add comments for complex logic
- Handle errors gracefully

## Testing

Before submitting:
1. Test the module in a development Odoo instance
2. Verify all features work as expected
3. Check for console errors
4. Test on different screen sizes

## Reporting Issues

When reporting issues, please include:
- Clear description of the problem
- Steps to reproduce
- Expected behavior
- Actual behavior
- Screenshots if applicable
- Odoo version
- Browser used

## Questions?

If you have questions, feel free to open an issue for discussion.
